<h1>JWipe - Active directory Home Lab</h1>


<h2>Description</h2>
In this transformative lab experience, I embarked on the intricate journey of establishing an Active Directory home lab environment utilizing Oracle VirtualBox. Through meticulous setup and continuous refinement, I immersed myself in the complexities of Active Directory and Windows networking, enriching my understanding with each configuration tweak. This hands-on approach not only deepened my comprehension but also instilled in me the confidence to navigate the multifaceted realm of network administration.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
